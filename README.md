# Linked List Programs in C

This document contains simple and clear **C programs** that demonstrate various types and operations of **linked lists**, including singly, doubly, circular, and header linked lists, as well as a program for **polynomial addition** using linked lists.

---

## 🔹 1. Basic Linked List (Static)
**Description:**  
A simple linked list with three nodes created manually and displayed.

**Key Points:**
- Uses static node creation.
- Demonstrates `next` pointer linking and traversal.
